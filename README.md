<h1 align="center">LocalDrop</h1>

<p align="center">Native macOS file sharing for the LocalSend ecosystem</p>

LocalDrop is a native macOS file-sharing app fully compatible with the [LocalSend](https://github.com/localsend/localsend) protocol. It lets you securely send files and text to nearby LocalSend-compatible devices over your local network, without requiring an internet connection.

This project was created to provide a more deeply integrated macOS experience, with a fully native interface, system-level behavior, and performance tailored specifically for the Mac.

It is built with gratitude to the original LocalSend project and its contributors, whose open protocol and open-source work made this app possible. LocalSend remains the foundation of the wider cross-platform ecosystem.

## Features

- Discover nearby devices on the same Wi-Fi network.
- Send files, folders, and text to any LocalSend peer.
- Receive files with optional PIN protection.
- Browse transfer history and reveal received files in Finder.
- Run from the menu bar with optional launch-at-login.
- Localize the UI into one of 17 supported languages.

## Requirements

- macOS 14 or later.
